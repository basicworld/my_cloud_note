# my_cloud_note
学习springmvc用的云笔记项目

本项目是对一个开源半成品项目的重构实现：
1.	项目思路来自开源项目
2.	原项目没有完全完成，这里把功能都补充完整
3.	原项目前端采用html，这里改成jsp，为了学习研究jsp
4.	本项目在功能上完全参照原项目，但是前后台的实现都是重构的。前台自己画原型、自己写html、css、js。后台整体架构不变，所有的实现代码都是重构的。

通过本项目可以初步掌握从需求、设计、开发、测试、部署的全流程，向全栈工程师迈进。尤其是掌握ssm框架。
